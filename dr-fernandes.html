Dr. Myra Fernandes

Principal Investigator, Cognitive Neuroscience Lab

Dr. Myra Fernandes is a leading researcher in the field of cognitive neuroscience, specializing in memory processes and their neural underpinnings. Her innovative research explores how various factors influence memory formation, consolidation, and retrieval, with a particular focus on drawing as a mnemonic technique.

Research Focus

Dr. Fernandes leads research investigating the cognitive and neural mechanisms underlying memory processes, with several key areas of focus.

Drawing and Memory

Dr. Fernandes has pioneered research on how drawing enhances memory for verbal information. Her studies demonstrate that drawing pictures of to-be-remembered information leads to superior memory compared to other encoding strategies.

This research has shown that drawing is effective across different populations, including older adults and individuals with various cognitive impairments. The findings suggest that drawing provides a rich and multifaceted encoding experience that facilitates later recall.

Emotion and Memory

Another significant area of Dr. Fernandes' research explores how emotional states and emotional content influence memory processes. Her work examines the neural systems that support emotional memory and how these systems interact with cognitive control processes.

Using behavioral, neuroimaging, and psychophysiological methods, this research has revealed important insights into how emotional arousal enhances memory formation and how individual differences affect emotional memory processing.

Attention and Memory

Dr. Fernandes investigates how attentional processes influence memory encoding and retrieval. Her research examines the effects of divided attention on memory performance and the neural mechanisms that support successful memory formation under different attentional states.

This work has practical implications for understanding how multitasking affects learning and memory in educational and everyday settings, as well as how attentional deficits contribute to memory problems in clinical populations.

Aging and Memory

A significant portion of Dr. Fernandes' research examines how memory processes change across the lifespan, with a particular focus on healthy aging and neurodegenerative conditions.

Her studies investigate age-related changes in different memory systems, compensatory mechanisms that support memory in older adults, and interventions that can enhance memory function in aging populations. This research contributes to our understanding of cognitive aging and informs strategies to maintain cognitive health throughout the lifespan.
