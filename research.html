Our Research

Exploring the neural mechanisms of memory, cognition, and perception through innovative neuroscientific approaches to advance our understanding of the human mind.

Research Overview

Our lab investigates the cognitive and neural mechanisms underlying human memory and related cognitive processes. We employ a range of methodologies including behavioral experiments, neuroimaging techniques, and computational modeling.

Behavioral Studies

Carefully designed experiments to measure cognitive processes and memory performance in controlled laboratory settings.

Neuroimaging

Using fMRI, EEG, and other brain imaging techniques to examine brain activity during memory encoding, storage, and retrieval.

Computational Modeling

Developing mathematical and computational models to formalize theories of memory and cognitive processes.

Research Areas

Memory & Attention

Our research examines how attention influences memory encoding and retrieval. We investigate the neural mechanisms that support selective attention and its impact on what information is stored in memory and subsequently remembered.

Current projects focus on:

  • How divided attention affects memory encoding and retrieval
  • The neural correlates of successful memory formation under different attentional states
  • The interaction between attention and emotional arousal in memory processes

Emotion & Memory

We study how emotional content affects memory encoding, storage, and retrieval. Our research examines the neural systems that support emotional memory and how these systems interact with cognitive control processes.

Key research directions include:

  • The effect of emotional valence and arousal on memory processes
  • How emotional memories are consolidated over time
  • Individual differences in emotional memory processing
  • The role of the amygdala and prefrontal cortex in emotional memory
Learn more about this research

Aging & Memory

Our research examines how memory and cognitive processes change across the lifespan, with a particular focus on healthy aging and neurodegenerative conditions.

Current investigations focus on:

  • Age-related changes in memory systems and their neural correlates
  • Compensatory mechanisms that support memory in older adults
  • Interventions to enhance memory function in aging populations
  • Early cognitive markers of neurodegenerative disorders

Computational Cognitive Neuroscience

We develop and apply computational models to understand the neural mechanisms underlying cognitive processes, with a focus on memory and decision-making.

Our computational approaches include:

  • Neural network models of memory encoding and retrieval
  • Bayesian models of learning and decision-making
  • Dynamical systems approaches to understanding neural information processing
  • Advanced analysis methods for neuroimaging data
Learn more about this research

Current Projects

Neural Mechanisms of Memory Consolidation

This project investigates how memories are consolidated during sleep and rest periods. We use EEG and fMRI to track memory reactivation during off-line periods and examine how this relates to subsequent memory performance.

Memory Sleep fMRI EEG

Principal Investigators: Dr. Myra Fernandes, Dr. James Wilson

Emotional Memory in Aging

This project examines how emotional memory changes with age. We investigate age-related differences in the processing of emotional stimuli and how these differences impact memory formation and retrieval.

Emotion Aging Memory Neuroimaging

Principal Investigators: Dr. Myra Fernandes, Dr. Elena Rodriguez

Neural Network Models of Recognition Memory

This project develops computational models of recognition memory, focusing on how neural networks can account for human recognition performance across different experimental conditions.

Computational Neural Networks Recognition Modeling

Principal Investigators: Dr. Michael Park, Dr. James Wilson

Drawing and Memory Enhancement

This project investigates how drawing can enhance memory encoding compared to other encoding strategies. We examine the cognitive and neural mechanisms that underlie the memory benefit of drawing.

Drawing Memory Encoding Behavioral

Principal Investigators: Dr. Myra Fernandes, Priya Patel
